# Overview

This is a repo containing a simple script for pass otp completions in nushell.

## How to install

1) Clone this repo
2) Place the path to `./pass_extensions_completions.nu` at the bottom of `config.nu`

Example
```nu
  # The entirety of your awesome config.nu file
  # ...
  # ...
  # ...
  # After setting any custom completer

  source path/to/pass_extensions_completions.nu
```
